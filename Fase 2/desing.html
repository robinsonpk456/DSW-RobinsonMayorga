Propósito del sitio:

Este sitio web tiene como propósito desarrollar la temática de Etiquetas Semánticas, CSS Grid Layout con el fin de definir su contenido y distribución en la página web para así dinamizar las estrategias de aprendizaje de los estudiantes de la universidad y obtener como resultado de la actividad una herramienta didáctica utilizando el leguaje HTML5 de acuerdo con los requerimientos del usuario y/o cliente.

Etiquetas Semánticas:

Las etiquetas semánticas ayudan a definir la estructura del documento y permiten que las páginas web sean mejor indexadas por los navegadores.

Una etiqueta se califica como semántica si tiene que ver el significado es decir si nos informa sobre su contenido. Por ejemplo la etiqueta SECTION nos dice que contiene una sección o capítulo dentro de la página.

CSS Grid Layout:

Es una técnica de hoja de estilo en cascada que le permite a los desarrolladores web crear diseños complejos y adaptables con mayor facilidad en todos los motores de busqueda.

CSS Grid Layout se destaca al dividir una página en regiones principales o definir la relación en términos de tamaño, posición y capa, entre partes de un control creado a partir de primitivas HTML.
